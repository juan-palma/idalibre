La historia oculta de un sitio web

No hay dos sitios idénticos con distinto precio. Si pagaste lo que vale un par de tenis, tu web lleva piezas recortadas. A veces está bien; el problema es no saber qué se quedó fuera.

Un día cualquiera

Cliente, 2:30 p. m.: “Me ofrecen la página en $2 500, ¿la igualas?”
Yo: “Claro. ¿Incluye velocidad, SEO y soporte?”
Cliente: …silencio.

Así arrancan ocho de cada diez proyectos que rescato: el precio es la portada; la trama cambia con cada atajo.
